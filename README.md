# Augmented Reality

Augmented Reality (almost) from scratch

## Install

```bash
git clone git@github.com:l4cer/augmented-reality.git
cd augmented-reality
python -m venv venv
./venv/Scripts/activate   # Windows
source venv/bin/activate  # Linux
pip install -r requirements.txt
```

## Execute

```bash
python main.py
```
